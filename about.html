About NeonHueMusic

NeonHueMusic is a dynamic, web-based music player inspired by Spotify, developed using HTML, CSS, and JavaScript. It provides users with an engaging and interactive music streaming experience, complete with a sleek UI, play/pause functionality, song lists, and real-time progress tracking.

This project was created as a part of my learning journey in web development. I aimed to combine functionality with aesthetic design, resulting in a responsive and immersive platform for music lovers. Every feature was designed with user experience in mind, including the animated playback interface, album art display, and playlist interactions.

The project also showcases my front-end development skills, especially in DOM manipulation, responsive design, and event-driven programming. I continue to improve it by adding features such as dynamic song loading, volume control, and playlist management.
